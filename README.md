A Flutter package to get zodiac from date

## Features

- returning zodiac as a String
- for now, function argument only support yyyy-mm-dd string format

## Usage
Example:
```dart
const zodiac = Zodiac().getZodiac('2000-01-01'); // Capricorn
```
